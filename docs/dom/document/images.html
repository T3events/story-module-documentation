document.images

document.images returns a collection of the images in the current HTML document.

Syntax

var htmlCollection = document.images;

Example

var ilist = document.images;

for(var i = 0; i < ilist.length; i++) {
    if(ilist[i].src == 'banner.gif') {
        // found the banner
    }
}
