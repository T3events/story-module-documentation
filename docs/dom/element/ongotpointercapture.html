element.ongotpointercapture

ongotpointercapture is an EventHandler property of the Element interface that returns the event handler (function) for the gotpointercapture event type.

Syntax

var gotCaptureHandler = target.ongotpointercpature;

Return value

gotCaptureHandler
The gotpointercapture event handler for element target.

Example

<html>
<script>
function overHandler(ev) {
 // Determine the target event's gotpointercapture handler
 var gotCaptureHandler = ev.target.ongotpointercapture;
}
function init() {
 var el=document.getElementById("target");
 el.onpointerover = overHandler;
}
</script>
<body onload="init();">
<div id="target"> Touch me ... </div>
</body>
</html>
