navigator.buildID

Returns the build identifier of the browser.

Syntax

buildID = navigator.buildID;

buildID is the build identifier of the application as a string. The build ID is in form YYYYMMDDHH.

Example

dump(window.navigator.buildID); 
// prints "2006090803" to the console

Note

Available in Firefox 2 (Gecko 1.8.1) and later. Implemented in bug 345993.

Specification

Not part of any public standard.
