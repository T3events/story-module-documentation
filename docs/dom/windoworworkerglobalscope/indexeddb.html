WindowOrWorkerGlobalScope.indexedDB

The indexedDB property of the WindowOrWorkerGlobalScope mixin provides a mechanism for applications to asynchronously access the capabilities of indexed databases.

Syntax

var DBOpenRequest = self.indexedDB.open('toDoList');

Value

An IDBFactory object.

Example

var db;
function openDB() {
 var DBOpenRequest = window.indexedDB.open('toDoList');
 DBOpenRequest.onsuccess = function(e) {
   db = DBOpenRequest.result;
 }
}
