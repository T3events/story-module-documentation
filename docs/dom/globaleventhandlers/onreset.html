GlobalEventHandlers.onreset

The GlobalEventHandlers.onreset property contains an EventHandler triggered when a reset event is received.

Syntax

window.onreset = funcRef;

Parameters

  • funcRef is a reference to a function.

Example

<html>
<script>
function reg() {
  window.captureEvents(Event.RESET);
  window.onreset = hit;
}
 
function hit() {
 alert('hit');
}
</script>

<body onload="reg();">
 <form>
   <input type="reset" value="reset" />
 </form>
 <div id="d"> </div>
</body>
</html>

Notes

The reset event is raised when the user clicks a reset button in a form (<input type="reset"/>).
