The ready readonly property of the FontFaceSet interface returns a Promise that resolves to the given FontFaceSet.

Syntax

fontFaceSet.ready.then(function(fontFaceSet) {
  // ...
});

Returns

A Promise that resolves to the given FontFaceSet.

Parameters

None.
