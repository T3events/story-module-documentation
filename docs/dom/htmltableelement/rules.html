HTMLTableElement.rules

The HTMLTableElement.rules property represents which cell borders to render in the table.

Syntax

HTMLTableElement.rules =
rules;
var
rules =
HTMLTableElement.rules;

Parameters

  • rules is a string with one of the following values:
    none
    no rules
    groups
    lines between groups only
    rows
    lines between rows
    cols
    lines between cols
    all
    lines between all cells

Example

// Turn on all the internal borders of a table
var t = document.getElementById("TableID");
t.rules = "all";

Specification

  • W3C DOM 2 HTML Specification
