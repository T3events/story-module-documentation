HTMLTableElement.tHead

The HTMLTableElement.tHead represents the table's <thead> element. Its value can be null if there is no such element.

Syntax

thead_element = table.tHead;
table.tHead = thead_element;

Parameters

  • th_el is a HTMLTableSectionElement.

Example

if (table.tHead == my_head_el) {
    // ...
}
