HTMLTableElement.summary

The HTMLTableElement.summary property represents the table description.

Syntax

HTMLTableElement.summary =
string;
var
string =
HTMLTableElement.summary;

Example

HTMLTableElement.summary = "Usage statistics";

Specification

  • W3C DOM 2 HTML Specification
