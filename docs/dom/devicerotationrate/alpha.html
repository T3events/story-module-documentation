DeviceRotationRate.alpha

This property indicates the rate of rotation around the Z axis -- in degrees per second -- in a DeviceRotationRate object.

Syntax

var alpha = deviceRotationRate.alpha;

This property is read-only.

Return value

alpha
A double indicating the rate of rotation around the Z axis, in degrees per second. See Accelerometer values explained for details.
