PresentationConnection.binaryType

When a PresentationConnection object is created, its binaryType IDL attribute MUST be set to the string " arraybuffer". On getting, it MUST return the last value it was set to. On setting, the user agent MUST set the IDL attribute to the new value.

Note: The binaryType attribute allows authors to control how binary data is exposed to scripts. By setting the attribute to " blob", binary data is returned in Blob form; by setting it to "arraybuffer", it is returned in ArrayBuffer form. The attribute defaults to " arraybuffer". This attribute has no effect on data sent in a string form.
