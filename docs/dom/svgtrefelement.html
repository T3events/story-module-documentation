SVGTRefElement

SVG tref DOM interface

The SVGTRefElement interface provides access to the properties of <tref> elements, as well as methods to manipulate them.

Interface overview

Also implement SVGTextPositioningElement, SVGURIReference
Methods None
Properties None
Normative document SVG 1.1 (2nd Edition)

Properties

The SVGTRefElement interface do not provide any specific properties.

The SVGTRefElement interface do not provide any specific methods.
