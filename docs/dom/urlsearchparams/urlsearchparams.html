URLSearchParams.constructor

This is an experimental technology
Because this technology's specification has not stabilized, check the compatibility table for usage in various browsers. Also note that the syntax and behavior of an experimental technology is subject to change in future versions of browsers as the specification changes.

The URLSearchParams() constructor creates and returns a new URLSearchParams object. Leading '?' are ignored.

Syntax

var URLSearchParams = new URLSearchParams(init);

Parameters

init Optional
A USVString or a URLSearchParams instance.

Return value

An instance of URLSearchParams.

Example

The following example shows how to create a URLSearchParams object from a URL string.

// or construct from window.location
let url = new URL('https://example.com?foo=1&bar=2'); 

let params = new URLSearchParams(url.search);
