HTMLHyperlinkElementUtils.hash

The HTMLHyperlinkElementUtils.hash property returns a DOMString containing a '#' followed by the fragment identifier of the URL. The fragment is not percent-decoded.

Syntax

string = object.hash;
object.hash = string;

Examples

// Let's an <a id="myAnchor" href="https://developer.mozilla.org/en-US/docs/HTMLHyperlinkElementUtils.href#youhou"> element be in the document
var anchor = document.getElementById("myAnchor");
var result = anchor.hash; // Returns:'#youhou'
