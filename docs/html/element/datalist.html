<datalist>

The HTML <datalist> element contains a set of <option> elements that represent the values available for other controls.

Content categories Flow content, phrasing content.
Permitted content Either phrasing content or zero or more <option> elements.
Tag omission None, both the starting and ending tag are mandatory.
Permitted parents Any element that accepts phrasing content.
Permitted ARIA roles None
DOM interface HTMLDataListElement

Attributes

This element has no other attributes than the global attributes, common to all elements.

Examples

<label>Choose a browser from this list:
<input list="browsers" name="myBrowser" /></label>
<datalist id="browsers">
  <option value="Chrome">
  <option value="Firefox">
  <option value="Internet Explorer">
  <option value="Opera">
  <option value="Safari">
  <option value="Microsoft Edge">
</datalist>

Result

Using the result

The <input> attribute name specifies the name of the control, which is submitted with the form data.

This form data is then available in the server in the http request: String myBrowser = request.getParameter("myBrowser");
