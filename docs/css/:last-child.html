:last-child

The :last-child CSS pseudo-class represents any element that is the last child element of its parent.

Syntax

:last-child { style properties }

Example

HTML Content

<ul>
  <li>This item is not limed.</li>
  <li>Also not this one.</li>
  <li>This item is! :)</li>
</ul>

CSS Content

li:last-child {
  background-color: lime;
}
