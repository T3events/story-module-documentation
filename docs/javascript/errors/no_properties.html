Errors: No properties

Message

TypeError: null has no properties
TypeError: undefined has no properties

Error type

TypeError.

What went wrong?

Both, null and undefined, have no properties you could access.

Examples

null.foo;
// TypeError: null has no properties

undefined.bar;
// TypeError: undefined has no properties
