dataView.setFloat32

The setFloat32() method stores a signed 32-bit float (float) value at the specified byte offset from the start of the DataView.

Syntax

dataview.setFloat32(byteOffset, value [, littleEndian])

Parameters

byteOffset
The offset, in byte, from the start of the view where to store the data.
value
The value to set.
littleEndian
Optional Indicates whether the 32-bit float is stored in little- or big-endian format. If false or undefined, a big-endian value is written.

Return value

undefined.

Errors thrown

RangeError
Thrown if the byteOffset is set such as it would store beyond the end of the view.

Examples

Using the setFloat32 method

var buffer = new ArrayBuffer(8);
var dataview = new DataView(buffer);
dataview.setFloat32(1, 3);
dataview.getFloat32(1); // 3
