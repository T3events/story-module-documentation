map.@@toStringTag

The Map[@@toStringTag] property has an initial value of "Map".

Property attributes of Map.prototype[@@toStringTag]
Writable no
Enumerable no
Configurable yes

Syntax

Map[Symbol.toStringTag]

Example

Object.prototype.toString.call(new Map()) // "[object Map]"
