string.toUpperCase

The toUpperCase() method returns the calling string value converted to upper case.

Syntax

str.toUpperCase()

Return value

A new string representing the calling string converted to upper case.

Description

The toUpperCase() method returns the value of the string converted to upper case. toUpperCase() does not affect the value of the string itself.

Examples

Using toUpperCase() 

console.log('alphabet'.toUpperCase()); // 'ALPHABET'
