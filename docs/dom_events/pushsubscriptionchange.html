pushsubscriptionchange

The pushsubscriptionchange event is fired to indicate that A PushSubscription has expired.

General info

Specification
Push API
Interface
PushEvent
Bubbles
No
Cancelable
No
Target
Service worker
Default Action
None

Properties

None.

Examples

self.addEventListener('pushsubscriptionchange', function() {
  // do something, usually resubscribe to push and
  // send the new subscription details back to the
  // server via XHR or Fetch
});
