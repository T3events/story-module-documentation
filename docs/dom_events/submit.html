submit

The submit event is fired when a form is submitted.

Note that submit is fired only on the form element, not the button or submit input. (Forms are submitted, not buttons.)

General info

Specification
HTML5
Interface
Event
Bubbles
Yes (although specified as a simple event that doesn't bubble)
Cancelable
Yes
Target
Element
Default Action
Varies (send the content of the form to the server).

Properties

Property Type Description
target Read only EventTarget The event target (the topmost target in the DOM tree).
type Read only DOMString The type of event.
bubbles Read only Boolean Whether the event normally bubbles or not
cancelable Read only Boolean Whether the event is cancellable or not?
