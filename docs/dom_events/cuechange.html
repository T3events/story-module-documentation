cuechange

The cuechange event fires when a TextTrack has changed the currently displaying cues.

See also

See https://www.whatwg.org/specs/web-apps/current-work/#event-media-cuechange

Browser Compatibility

This event isn't currently implemented in Firefox. See bug 1033144 for status.
