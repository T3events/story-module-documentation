surfacescale

The surfaceScale attribute represent the height of the surface for a light filter primitive. If the attribute is not specified, then the effect is as if a value of 1 were specified.

Usage context

Categories None
Value <number>
Animatable Yes
Normative document SVG 1.1 (2nd Edition) : feDiffuseLighting
SVG 1.1 (2nd Edition) : feSpecularLighting

Example

Elements

The following elements can use the surfaceScale attribute

  • <fediffuselighting>
  • <fespecularlighting>
