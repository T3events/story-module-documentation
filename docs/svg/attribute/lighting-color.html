lighting-color

The lighting-color attribute defines the color of the light source for filter primitives elements <fediffuselighting> and <fespecularlighting>.

Usage context

Categories Presentation attribute
Value currentColor | <color> | <icccolor> | inherit
Initial value white
Animatable Yes
Normative document SVG 1.1 (2nd Edition)

Examples

Elements

The following elements can use the lighting-color attribute

  • <fediffuselighting>
  • <fespecularlighting>
