GET

The HTTP GET method requests a representation of the specified resource. Requests using GET should only retrieve data.

Request has body No
Successful response has body Yes
Safe Yes
Idempotent Yes
Cacheable Yes
Allowed in HTML forms Yes

Syntax

GET /index.html

Specifications

Specification Title
